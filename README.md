# Cucumber tool demo

This repository contains demonstrations of software testing tools, specifically focusing on **Cucumber** for Behavior-Driven Development (BDD).

## About Cucumber

Cucumber is a software testing tool that supports Behavior-Driven Development (BDD). It allows expected software behaviors to be specified in a logical language that customers can understand (Gherkin). Key features include:

- **Gherkin Syntax**: Tests are written in plain English using `Given`, `When`, `Then` steps.
- **Collaboration**: Acts as a bridge between technical and non-technical team members (developers, QA, and business analysts).
- **Integration**: Works seamlessly with Java, Selenium, JUnit, and other tools to automate UI and API tests.

## Projects included:

- **SauceDemo** (formerly CucumberDemo) - A foundational Cucumber integration demo.
- **OrangeHRM-Automation** - Demonstrates page object models and automated scenarios using Cucumber.
- **ParaBank** - Additional test scenarios focused on banking features using BDD.
